Webby API

Для работы с Webby API необходимо получить токен.

С помощью нашего API вы сможете:

  1. Сообщения:
    • Получение своих сообщений в формате json
    • Отправление сообщений
  2. Профиль:
    • Получение данных своего профиля в формате json
    • Изменение данных своего профиля
    • Получение общедоступных данных о других пользователях
  3. Объявления:
    • Получение всех объявлений в формате json
    • Получение объявлений других пользователей в формате json
    • Получение своих объявлений в формате json
    • Изменение своих объявлений
    • Удаление своих объявлений
